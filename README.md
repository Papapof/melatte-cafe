# Portafolio de software

Esta carpeta contiene tres proyectos separados preparados para repositorios independientes:

- `registro-ingresos/` — aplicación de registro diario de ingresos y retiros.
- `app-melatte/` — aplicación de registro de ventas para cafetería.
- `software-actualizado/` — aplicación de control de estacionamiento y transacciones.

Cada proyecto tiene su propia carpeta y un `README.md` con la descripción y cómo usarlo.

> Nota: Los archivos ZIP originales se han conservado en la raíz como copia de los paquetes originales, pero los proyectos ya están descomprimidos y organizados en carpetas independientes.
